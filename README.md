# 🌿 FreshVision AI

AI-powered fruit freshness detection system using Deep Learning, EfficientNet, and Streamlit.

## 🚀 Features

- 🍎 Detects Fresh vs Rotten fruits
- 🧠 EfficientNet-B0 deep learning model
- ⚡ Real-time prediction
- 📸 Upload image or use webcam
- 🛡 Out-of-Distribution (OOD) rejection
- 🎨 Modern futuristic Streamlit UI

---

## 🧠 Supported Fruits

- Apple
- Banana
- Orange

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/jainleo2006-creator/freshvision-ai.git
cd freshvision-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```
